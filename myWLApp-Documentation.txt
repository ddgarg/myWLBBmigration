 1. It is a Kraken-js based app. Kraken is a middleware which helps in scaffolding.
   Find more information about Kraken here:
   site 	- http://krakenjs.com/
   github 	- https://github.com/krakenjs/kraken-js
   
2. Configurations: https://github.com/krakenjs/confit  
3. Fetching value from config file:
http://stackoverflow.com/questions/24612096/how-to-get-a-config-from-within-a-controller-require-it-from-a-controller
https://github.com/krakenjs/kraken-js/issues/248

for Kraken < v1.0
var config = require('nconf');

var shouldHide = config.get('middleware:myModule:hideWidget');

if (shouldHide === true) {
  // ... do something ...
}

for Kraken > v1.0
// ./routes/secure.js
'use strict';

module.exports = function (req, res) {
  var shouldHide = req.app.kraken.get('middleware:myModule:hideWidget');
  if (shouldHide === true) {
    // ... do something ...
  }
};

4. While adding any new node module make sure to add in package.json use ==>> eg. npm install --save fbgraph. it would automatically add to package.json
